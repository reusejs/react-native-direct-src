# Inline
Inline is a layout container which alligns it's children horizontally (inline) and provide constant margin around them.

> Use of Inline is discraged. Use `Stack` with direction="horizontal" instead.

# Usage
```js
import { Box, Badge } from 'react-native-design-system';

<Inline space="xxlarge">
  <Badge onPress={() => this.handlePress(id)}>
    10
  </Badge>
  <Badge onPress={() => this.handlePress(id)}>
    50
  </Badge>
  <Badge onPress={() => this.handlePress(id)}>
    677
  </Badge>
  <Badge onPress={() => this.handlePress(id)}>
    9999
  </Badge>
</Inline>
```

# Space
Customize spacing of each child from one another. `space` prop accepts one of the given values: `none`, `xxsmall`, `xsmall`, `small`, `medium`, `large`, `xlarge`, `xxlarge` and provide constant spacing between each children.
