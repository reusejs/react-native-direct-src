# Box
Box is a simple layout container which accepts a child and provide padding around it. The child is centered by default.

# Usage
```js
import { Box } from 'react-native-design-system';

<Box space="large">
  <Button>
    Press Me
  </Button>    
</Box>
```

# Props
<Props of={Box} />

# Space
Customize padding for the child. `space` prop accepts one of the given values: `none`, `xxsmall`, `xsmall`, `small`, `medium`, `large`, `xlarge`, `xxlarge`

<Preview>

</Preview>

# Background
You can also pass a `background` color. Color string should be present inside `brandColor` of theme.
