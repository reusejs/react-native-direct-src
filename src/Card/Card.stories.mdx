# Card
Card is a wrapper component with a small padding. It included center, left and right aligned card to auto align childrens. Use `Box`, `Stack` or `Inline` when making a layout.

# Usage
```js
import { Card, Button } from 'react-native-design-system';

<Card>
  <Button onPress={() => this.handleChange())}>
    Press Me
  </Button>
</Card>
```

# Type
There are 3 types of alignment for children: `left`, `center` and `right`. You just have to pass `align` prop which accepts a string from three choices.

# Padding
Padding can be applied horizontally and vertically only.
* First button is getting `horizontal` prop which makes vertical padding `0`.
* Second button is getting `vertical` prop which makes horizontal padding `0`.

# Row
Default alignment of childrens is `column` which can be changed to `row` by passing `row` prop. It's a boolean value. This is totally a personal use-case and you can always customize the view with `style` prop.
