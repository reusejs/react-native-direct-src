# CheckBox
Use `CheckBox` when looking for yes or no answers.

# Usage
```js
import { CheckBox } from 'react-native-design-system';

<CheckBox checked={this.state.checked} onPress={() => this.setState({ checked: !this.state.checked })}>
    Hello CheckBox
</CheckBox>
```
